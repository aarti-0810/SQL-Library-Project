# Digital Library Audit System

## Description

This project is a SQL-based system to manage books, students, and issued books in a library.

## Features

* Tracks issued and returned books
* Identifies overdue books (not returned within 14 days)
* Finds most popular book categories
* Marks inactive students (no activity for 3 years)

## Technologies Used

* MySQL

## Files

* library_project.sql → contains all queries and table creation

## How to Run

1. Open MySQL Workbench
2. Run the SQL script
3. View outputs using SELECT queries
